# Custom Components Example

This page demonstrates how to use custom components in your MDX files.

## Built-in Components

These components are automatically available in all MDX files:

<Alert type='info'>This is an info alert using the built-in Alert component.</Alert>

<Alert type='warning'>This is a warning alert.</Alert>

<Alert type='error'>This is an error alert.</Alert>

<Alert type='success'>This is a success alert.</Alert>

<Button>Click me!</Button>

<Card>
  <h3>This is a card</h3>
  <p>Cards are great for organizing content.</p>
</Card>

## Custom Components

These components come from your `mdx-components.js` file:

<CustomAlert type='info'>This is a custom alert component with a different style.</CustomAlert>

<CustomAlert type='warning'>Custom alerts have a left border design.</CustomAlert>

<InfoCard title='Information Card'>
  This is a custom info card component that wraps content in a nice container. You can put any content here, including: - Lists - **Bold text** - `Code
  snippets` - And more!
</InfoCard>

<InfoCard>This info card has no title, just content.</InfoCard>

## Customization

To add your own components:

1. Create a `mdx-components.js` file in your project root
2. Export your custom components
3. They'll be automatically available in all MDX files

Example:

```javascript
// mdx-components.js
export const MyComponent = ({ children }) => {
  return <div className='my-custom-style'>{children}</div>
}
```

Then use it in any MDX file:

```mdx
<MyComponent>This content will be styled with your custom component.</MyComponent>
```

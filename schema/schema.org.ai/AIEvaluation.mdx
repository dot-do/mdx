# AIEvaluation

**AIEvaluation** documents the performance evaluation of an AI model on specific datasets and metrics.

## Examples

```yaml
$type: AIEvaluation
model: claude-3-opus
dataset: mmlu-benchmark
metrics:
  - name: accuracy
    value: 0.867
  - name: f1-score
    value: 0.854
score: 0.867
evaluatedAt: 2025-10-01T00:00:00Z
```

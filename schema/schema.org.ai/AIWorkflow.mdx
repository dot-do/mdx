# AIWorkflow

An **AIWorkflow** represents a multi-step process involving AI agents, tools, and human collaboration.

## Examples

```yaml
$type: AIWorkflow
name: Content creation workflow
steps:
  - research-topic
  - generate-outline
  - write-draft
  - review-and-edit
  - publish
agents:
  - research-agent
  - writer-agent
  - editor-agent
estimatedTime: 3600
successRate: 0.88
```

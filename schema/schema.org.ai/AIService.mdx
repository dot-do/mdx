# AIService

**AIService** represents an API service that provides AI capabilities to applications.

## Examples

```yaml
$type: AIService
name: OpenAI API
apiEndpoint: https://api.openai.com/v1
authentication: api-key
capabilities:
  - text-generation
  - embeddings
  - image-generation
models:
  - gpt-4
  - dall-e-3
rateLimit: 10000
```

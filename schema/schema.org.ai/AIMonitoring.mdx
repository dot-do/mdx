# AIMonitoring

**AIMonitoring** represents ongoing monitoring of AI models in production to track performance, detect issues, and ensure quality.

## Examples

```yaml
$type: AIMonitoring
model: content-moderation-model
metrics:
  - accuracy
  - latency
  - throughput
  - error-rate
alerts:
  - type: accuracy-drop
    threshold: 0.05
    action: notify-team
period: realtime
lastChecked: 2025-10-02T10:30:00Z
```
